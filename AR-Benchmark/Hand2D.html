ZJU Hand2D Dataset


Introduction

We introduce a new dataset for testing hand keypoints detection, including different postures of the left and right hands.

You can download our dataset from here.


examples


  • Dior format

    • The dataset was collected using binocular cameras, with folders 0 and 1 representing the data corresponding to each camera. The "_left" and "_right" suffixes represent the left and right hands, and the ground truths of each image are saved in a JSON file with the same subdirectories and name:
