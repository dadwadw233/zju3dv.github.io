    The format for Android phone / iPhone is: 

              
          eg: data/Android phone/A0/android.tar.gz

          A0
          |--camera                               |
          |   |--sensor.yaml                      |
          |   |--data.csv                         |
          |   |--images                           |
          |       |--17896609415000.png           |
          |       |--17896643028000.png           |
          |       `--...                          | 
          |                                       |
          |--imu                                  |
          |   |--sensor.yaml                      |
          |   `--data.csv                         |
          |                                       |
          |--attitude                             |
          |   |--sensor.yaml                      |
          |   `--data.csv                         |
          |                                       |
          `--sensor                               |
              |--bluetooth.csv                    |
              |--mf.csv                           |
              |--gps.csv                          |
              `--...
